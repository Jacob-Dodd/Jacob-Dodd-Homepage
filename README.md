## Basic Project structure

Able to have a very simple project structure thanks to Next.js
```
$PROJECT_ROOT
│   # Page files
├── pages
│   # React component files
├── components
│   # Custom & Non-react modules
├── lib
│   # Static files for the 3D model and all our Images, etc
└── public
```
